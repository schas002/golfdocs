- - dash, minus - subtraction

Definition 1

Subtract two numbers. - in Python.[1]

Definition 2

Filter a collection on absence from another collection (both one of string, list, set, dictionary). Uses not in.[1]

Definition 3

Cast a number (the first argument) to a collection's (the second argument's) type (one of string, list, set, dictionary), then filter on absence.[1]

Definition 4

Cast a number (the second argument) to a collection's (the first argument's) type (one of string, list, set, dictionary), then filter on absence.[1]

Synopsis

-<minuend><subtrahend>
Where <minuend> and <subtrahend> depend on the definition.

Example

-3 2
-,1 2,1 3
 -1,1 2
